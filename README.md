nkcv
====

* nkcv は DMM が配信しているブラウザゲーム「艦隊これくしょん」のための専用ブラウザーです。
* 鋭意開発中です。

主な機能
----

* ゲーム画面の表示サイズを変えられます。
* ゲーム画面のスクリーンショットを撮影できます。
* 所属艦娘一覧を表示し、様々な条件で並び替えできます。
  * SQL が書ける方は、SQL でも検索できます。
* 遠征の残り時間、入渠中、建造中の艦娘の状況を一覧できます。

動作環境
----

* Windows (64bit 版のみ)
* macOS
